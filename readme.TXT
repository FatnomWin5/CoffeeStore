Программа позволяет выбрать и заказать кофе, также можно посмотреть состав в виде картинки и есть меню для бронирование стола.

Программа удовлетворяет следующим требованиям к содержанию проекта:
1)Тема для приложения выбрана студентом самостоятельно.
2)Приложение создано в среде Microsoft Visual Studio с нуля и имеет графический интерфейс WPF.
3)Приложение имеет многокомпонентную архитектру. Оно содержит запускаемый проект CoffeeStore и библиотеку dll(Library).
4)Наличие хранилища данных в виде коллекции. Постоянное хранилище данных в txt файле "Types.TXT".
5)Наличие методов для работы с этим хранилищем.
6)Использование интерфейсов (ICoffee).
7)Использование свойств.
8)Использование LINQ (MainWindow.xaml.cs строка 64).
9)Работа с файлами (чтение данных из файла).
10)Каждый метод снабжён комментарием.
11)Еще несколько окон(Table, СompositionWindow), помимо главного, передача информации между окнами.
